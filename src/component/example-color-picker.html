ColorPicker

A configurable web component for color selection with optional opacity control

1. Default Configuration (with opacity)

16 hue columns, 5 rows, 16px cells, opacity enabled.

Click a color to see details here

2. Minimal (no opacity, 8 hues)

A smaller picker: 8 hue columns, 4 rows, 14px cells.

3. Grayscale Only (no hues)

0 hue columns, just the grayscale + opacity.

4. Size Variations

Compact (8 cols)

Quick picking in tight spaces

Standard (16 cols)

Default balanced size

Full Spectrum (32 cols)

Detailed color selection

5. Programmatic Access

Get color values from JavaScript.

API Reference

Attributes

allow-opacity (boolean, default: false)
Show/hide the opacity picker column.
hue-columns (0-64, default: 16)
Number of hue columns. Set to 0 to show only grayscale.
rows (2-6, default: 5)
Number of rows, which affects both opacity and hue pickers.
cell-size (number, default: 16)
Size of each color cell in pixels.

Methods

getSelectedColor()
Returns {h, s, l, a} object
getSelectedHex()
Returns hex string, e.g. "#ff0000"
getSelectedRgba()
Returns rgba string, e.g. "rgba(255, 0, 0, 1.00)"

Events

color-picked
Fired when a color is clicked. Detail includes:
{ h: number, // Hue (0-360) s: number, // Saturation (0-100) l: number, // Lightness (0-100) a: number, // Opacity (0-100) hex: string, // e.g. "#ff0000" rgba: string // e.g. "rgba(255, 0, 0, 1.00)" }
